Contexto: Estamos construindo o site SmartVex na pasta principal tem uma imagem de como deve ser o design do site use isso, continuando quero um SaaS complexo de processamento de vídeo em Next.js 14+ (App Router) e FastAPI.
​Missão: Atue como um Engenheiro Full-Stack e automatize a geração da fundação do projeto. Não quero explicações básicas, quero o código estrutural e scripts de automação.
​1. Automação da Estrutura de Pastas (Shell Script)
​Gere um script .sh para Linux que crie automaticamente a árvore de diretórios do monorepo:
​/frontend: Estrutura Next.js com pastas (app), components/ui, hooks, services e store.
​/backend: Estrutura FastAPI com api/v1, core/config, models, workers (Celery) e utils/ffmpeg.
​/docker: Arquivos Dockerfile e um docker-compose.yml que suba o Next.js, FastAPI, Redis e Postgres de uma vez.
​2. Arquitetura de Processamento Assíncrono (O Coração)
​Automatize a lógica do Video Enhancer (Melhorador de Qualidade) no backend. Escreva uma função em Python que:
​Receba o caminho do vídeo.
​Detecte automaticamente o número de threads do processador (otimizado para Xeon).
​Execute o comando FFmpeg com filtros de: unsharp (nitidez), hqdn3d (denoising) e eq (contraste/brilho).
​Utilize WebSockets para enviar o progresso (frame/total_frames) em tempo real para o frontend.
​3. Automação do Frontend (Next.js + Tailwind)
​Gere o componente principal Dashboard.tsx para o SmartVex.
​Estilo: Ultra Dark Mode (Preto absoluto e cinza escuro).
​Funcionalidade: Um sistema de abas ou cards para as 6 funções (Corte, Legenda, Compressão, Conversão, Áudio, Enhancer).
​Automatize o estado de upload: Quando o usuário arrasta um vídeo, o componente deve disparar automaticamente o upload para a rota /api/v1/upload e abrir o monitor de progresso.
​4. Banco de Dados e Modelos (SQLAlchemy)
​Gere o modelo de dados para a tabela VideoJob. Deve conter: id, user_id, original_name, status (queued, processing, completed, failed), tool_used (qual das 6 funções), e output_url.
​5. Script de Inicialização
​Crie um arquivo Makefile ou package.json raiz que, com um único comando (make dev ou npm run dev:all), inicie simultaneamente o frontend Next.js, o servidor FastAPI e o worker do Celery.
​Instrução Final: Foque em código pronto para produção. Use tipagem estrita com TypeScript e Pydantic. Comece gerando o script de criação de pastas e o Docker Compose.
